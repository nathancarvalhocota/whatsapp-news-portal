# Front-end Next.js — WhatsApp News Portal

Portal público de notícias sobre o ecossistema WhatsApp.

## Responsabilidades

- Exibição de artigos publicados (SSR/SEO)
- Página inicial com listagem de artigos
- Página de artigo individual
- Página de categorias
- SEO técnico (meta tags, sitemap, structured data)

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Como rodar

```bash
# Copiar variáveis de ambiente
cp .env.example .env.local

# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

O front-end sobe por padrão em `http://localhost:3000`.
